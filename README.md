# ASX Stock Market Visualizer

A professional, realtime stock screener for the Australian Securities Exchange (ASX). This project tracks over 2,000+ stocks with daily automated updates.

**Live Demo:** https://asx-visualizer.vercel.app/

## Features
- **Daily Automation:** GitHub Actions fetches fresh data every market close (6:30 PM AEDT).
- **Deep Metrics:** Screen by P/E, RSI, Market Cap, Dividend Yield, and 50+ other metrics.
- **Interactive:** Client-side filtering, sorting, and sticky columns for easy analysis.
